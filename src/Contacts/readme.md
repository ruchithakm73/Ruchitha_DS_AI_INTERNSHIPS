Mini Project 1 — EDA Notebook
=================================

Overview
--------
This folder contains the exploratory data analysis notebook `MiniProject1_EDA.ipynb` which analyzes `customer_analytics.csv`.

How to open
-----------
- Open `MiniProject1_EDA.ipynb` in Jupyter Notebook, JupyterLab, or VS Code's notebook editor.

Dependencies
------------
Install common data-science packages if you don't already have them:

```
pip install pandas numpy matplotlib seaborn plotly jupyter
```

Notes
-----
- The notebook includes data cleaning and visualization steps — run cells top-to-bottom.
- If the CSV is not in the parent folder, update the file path in the notebook's first cell.
